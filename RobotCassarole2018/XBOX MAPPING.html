# Driver 1 (Drivetrain) - XBOX Controller

| Input | Type | Function | Description |

| LY | Axis | Fwd/Rev | Controls forward and reverse motion of robot |

| RX | Axis | LF/RT | Controls rotational motion of the robot |


# Driver 2 (Operators)- XBOX Controller

| Input | Type | Function | Description |

| LY | Axis | Left Winch | Controls Left Winch |

| RY | Axis | Right Winch | Controls Right Winch |

| LBumper | Button | Raise Climber* | |

| RBumper | Button | Elevator Mode | When held down allows for continuous elavator control |

| L Trigger | Axis | Lower Elevator | Lowers Elevator |

| R Trigger | Axis | Raise Elevator | Raises Elevator |

| A | Button | Intake | Activates Intake |

| X | Button | Eject | Ejects Cube |

| POV UP | POVHAT | POS Scale High | Raises Elevator to the scale's highest position |

| POV DN | POVHAT | POS Bottom | Lowers Elevator to Bottom position |

| POV RT | POVHAT | POS Switch | Raises/Lowers Elevator to Switch's position |

| POV LF | POVHAT | POS Exchange | Raises/Lowers Elevator to Exchange's Position |

| Start | Button | Intake Overide | Allows Operator to overide intake |

| LF BTN | Button | Pltfrm Release | Releases Platform Latch |

*Waiting for confirmation
